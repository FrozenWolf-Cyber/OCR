How to use ?

  All you have to do is just upload the scanned document in either image or pdf format and the rest will be done by us. 
  To annotate, you can either modify the classified labels, links and text generated during file conversion or just upload a new document and create your own labels and links

How does it work ?

 We use CRAFT network to detect sentences, Tesseract for word recognition, Faster-RCNN model for label classification, Siamese Neural Network for linking related sentences and keys
